DeleteNode 

case class DeleteNode(stmt: CfgStmtNode) extends OptimizationAction with Product with Serializable

Represents deletion of CfgStmtNode; every successor and predecessor of the node gets reconnected to each other Modifications performed:

  • CFG - node is removed from CFG and every successor and predecessor of the node gets reconnected to each other
  • contexts - context of the node is deleted
