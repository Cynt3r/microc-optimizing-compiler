  13.  class SemanticAnalysis extends Analysis[Declarations]

    An analysis that checks microC declarations.

    An analysis that checks microC declarations.

    Concretely, it checks the following:

    • Use of an undeclared identifier
    • Duplicate identifiers Note that while microC uses lexical scoping, for simplicity, we assume that each identifier appear is declared only once. This includes function names and variable names.
    • Duplicate record field names
    • Assignment to a function

    The result is a map from an identifier to its declaration.
