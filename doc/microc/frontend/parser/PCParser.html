microc.frontend.parser

PCParser 

class PCParser extends StringParsers with Parser

A parser combinator implementation of the microC grammar using a PEG-style parser.

Each of the public lazy val corresponds to one of the grammar non-terminal.

The microc.frontend.parser.Parser can be mixed in directly as the parser is stateless.
